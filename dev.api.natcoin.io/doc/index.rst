
Fusio
=====

Fusio is an open source API management platform which helps to build and manage 
RESTful APIs. The documentation covers the :ref:`installation` process, how to 
manage Fusio through the :ref:`backend`, how to use Fusio with 3rd party 
:ref:`consumer` applications and how to build reusable :ref:`adapter` components.
